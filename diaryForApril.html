4月日记

2017-4-3

前言

打算4月份进行一次体系知识的学习和总结（用小伙伴的话说：狗血的4月），我将尝试和我的小伙伴一起彼此监督的进行艰苦和快乐的4月共同学习之旅。这篇文章用于记录这段时间发生的事和我的想法，将以每天一更的形式记录4月份的收获与反思。

第一天

小伙伴在我旁边早读，挺好的，我挺喜欢这样。但感觉她不是很习惯。我希望小伙伴放开自己的学习状态，让她适应我在她旁边自习的节奏，避免我对她学习节奏造成的干扰。
我需要在她拿不定注意的地方，告诉她我的感受。
我希望我能准备好安全技术领域和容器技术领域的知识储备。并在课内和课外的学习中都能有所收获。
我将通过刷实习相关的题目提高自己面试安全技术岗位的面试能力。

第二天

今天心情很好，效率不错，跟队友的合作也很愉快。
开始动手编写和理解 Dockerfile 的语法，复习了一下yaml的语法，对 docker-compose 也有一定涉猎了，并且今天在自己的 aws 服务器上搭建了 git 私人仓库，接下来打算试试基于 git 服务器的自动化构建。
速度有点慢，主要还是自己心态不够踏实。有一点可以肯定，跟着视频踏实地打一遍代码才是最有效的学习途径。
发现自己有时太随性，太小气，像个孩子。待人处事，图彼此开心愉快，凡小事能让他人开心，自己吃点亏也是无妨。

第三天

一上课就手软。
学校的教学安排会影响我的学习状态。我需要调整一下自己的心态。
感冒了，一直在流鼻涕，昨天的收获有点少。也没怎么阅读。好在刷了刷题。
7号要开始京东的笔试，我得多刷一些面试题。
安全技术和 linux 容器技术将是我的切入点。

第五天

可以肯定,我找一个小伙伴一起自习是对的.我需要行动起来.
这个月我的目标是准备实习.并找到暑假的实习工作.关于实习,我其实还是处于想而不是行动的状态.
今天下午在反思我和一位女生之间的关系,不合适,得分开.今后面对感情,不要过早的表达,感情基础需要共同的生活,爱好,工作,信仰等任意一种可以创造共同时间的契机.条件不成立,谈感情为时尚早.
我的小伙伴现在比较忌讳男女关系,我要在关系的尺度上有所权衡.她是我的朋友,事情帮忙的尺度要有所控制.
英语早读是个好习惯,可以试着保持.
我需要在学习和提高上找到更多同行的小伙伴

第六天

我在思考如何坚持一个持续的,高投入和产出的学习方式和心态.
我在个人投入上相对保守,这意味着在产出上也会相对有限.这个月是实习准备月,我所应该准备的是一个高投入的姿态。
我认为，高投入的学习方式和心态实际上并不存在，存在的是模仿和习惯。即推动自己成为高投入的人的出发点是生活惯性（习惯）。当然，这里涉及到一个心态，就是，如果高投入的目的是为了高产出，这就需要模仿的力量，如果有一个人完成过你定下的目标，那么，我们可以模仿他。模仿的力量意味着可实现，有目标，在行动。

第七天

我不习惯主动的去约朋友出来自习。
昨天我没有约队友出来自习，下午开始玩游戏一直玩到了今天早上。
我的学习惯性一般。如果没有小伙伴一起学习，我可能会学着学着发呆玩起游戏来。
精神压力非常大（非常想逃避，想换个环境，精神接近崩溃）的时候玩玩游戏还是挺好的，可以有效的放松我的神经。但因为没有目标，因为失落而玩游戏则没有必要的。失落，没有目标，就去找有目标，不失落的人，跟他们闲聊也好，联彼此最近的状态也罢。最好不要通过游戏麻痹自己。
学习的结果固然重要，学习的过程是否恰当也很重要。
实习，能力水平是你要知道，重点是态度。
实习，需要合时宜的请教相关方面能力更强的的人。
实习，需要自己定个方向和目标。否则只能跟着别人的目标走。
最后提醒自己一点，合理可行的事，只要是以一种合理的方式去实践，我都可以去做。我需要主动点去试错。
早起读英语挺好的，可以让自己的生活节奏不会太乱，并提高自己阅读英语文章的能力，听别人朗读久了，自己英文断句的能力也能得到提高。

第八天

今天跟之前的英语老师谈了一下关于家庭方面的话题，很舒服。下次有机会可以继续跟她聊聊。
今天没有约队友一起自习。但我找到了自己的学习节奏。这挺好的。
如果我现在某个方面可以更快的进步或者找到自己前进的节奏（目标），请找一个这方面的小伙伴，然后请教他。
最近有点找不到目标。我所想要的无非是相关的实习，要做的无非是根据相关的目标制定目标然后前进。

第九天

我开始不知道接下来要如何开展后面的工作。没有目标。或者说我的目标不够坚定，不是发自我的内心的想法，isn't derived from heart。
第三天没有约队友一起自习。

第十天

今天给自己定个小目标。 把 kubernete 的教程过一遍。

第十二天

这两天玩游戏有点失控，不好。我需要再找一个一起自习的小伙伴，队友最近一直没有出现。
这段时间阅读有点少，可以在空闲的时间补补。
我对游戏的观念没有错，但在自控方面需要有一定的加强。

第十三天

今天决定采纳云计算王亚东老师的建议，开始刷网络课，并以获得证书为是否完成课程的标准
昨天阅读了插入排序和合并排序的 python 源代码，对 mergeSort 其采用的递归的思维方法我有点吃惊，毕竟通过几行代码就实现的一个现实数学模型，我很佩服。
今天继续数据结构的公开课，并收集相关算法的实现源码，会整后可以考虑上传到 git 上与他人分享。

第十四天

早上起来有点烦，所以打了会游戏。

今天遇到了一位很文艺的小伙伴，开心伴着吃惊，子卉，有趣的名字，有趣的人。她是这么的文艺，这么的有趣，我都有点克制不住自己的喜爱。我喜欢有趣的人，有趣的物，难以克制。如果有机会，真希望我能更好的了解她，了解她的喜好，我的求知欲，好奇心，让我忍不住想靠近她。
准备开始提高自己的阅读量了，并在豆瓣上记录自己看过的电影和书籍。
